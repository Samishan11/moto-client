╔════════════════════════════════════════════════════════════════════════════╗
║                      NAVIGATION STATE MACHINE                              ║
╚════════════════════════════════════════════════════════════════════════════╝

    Initial State
         │
         ▼
    [isLoading=true]
         │
         ├─ Check SecureStore for token
         │
         ▼
    [isLoading=false]
         │
         ├─ Token found?
         │
    ┌────┴─────┐
    │           │
   YES         NO
    │           │
    │           ├─ isSignout=false
    │           │  userToken=null
    │           │
    │           ▼
    │       [AuthStack]
    │       (Splash → Welcome → Login → Permissions)
    │
    ├─ isSignout=false
    │  userToken=<JWT>
    │
    ▼
[AppStack]
(MainTabs + Modal + FullScreenModal)

After Login:
    Login Screen → Dispatch SIGN_IN
                       │
                       ├─ Save token to SecureStore
                       ├─ Update userToken in state
                       │
                       ▼
                  AuthStack replaced with AppStack
                  → Lands on Permissions Screen
                  → Then navigates to Home
